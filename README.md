# CUT-RUN_normalisation
Code for the paper _"Benchmarking normalisation methods for differential binding analysis in CUT&amp;RUN"_. 

The code that processed the raw files, conducted the differential analysis on the BRG1 and ER experiments, simulation experiments for the Henikoff datasets and the code for figure generation are available here.



Contents: 

BRG1_dataset 

  - 01_raw_files_processing - contains workflow to process raw fastQ files to BAM and peak files. 
  - 02_normalisation_method_testing - code to generate scale factors and conduct differential analysis using the various normalisation methods  
  - 03_figure_generation - code to generate figures displayed in paper
